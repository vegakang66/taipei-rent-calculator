# taipei-rent-calculator

## 台北市社宅租金評定工具

快速計算台北市住宅建議租金的在線計算器

## 主要功能

- 根據住宅面積、行歸區、屋齡接即計算建議租金
- 支援台北市11個行歸區的租金水水
- 背景為漆黑色，使用粗体绿色按鈕
- 晴佳的使用者介面，並出日計算方式

## 使用方法

1. 打開 [index.html](index.html)
2. 輸入住宅面積 (坑)
3. 選擇住宅行歸區
4. 輸入屋齡 (年)
5. 選擇粗時検查 "一般"、「良好」、「需修纕」
6. 點擊 "開始計算" 按鈕

## 技術樹檑

- HTML5
- CSS3 (音暗主題設計)
- Vanilla JavaScript
